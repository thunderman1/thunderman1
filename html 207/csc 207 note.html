What is html?

html means hypertext markup language.Html is a markup language and not a programming language.html annotate text so that the browser can do the interpretation.It is the current language of the web.

Structure of the html tag

Tag usually begins with <>

container tags: is any tags that has beginning and ending.

Self closing tag or empty tag: you only open them but you donot close them.e.g Hr tag or br tag

Tool needed for web pages

  • Text Editor
    • This is an application used to produce plain text files.Text editor can also be used to edit programming source code.e.g notepad++,sublime text,Atom,Dreamweaver.
  • Web browser
    • This is a software that render web pages.or it is also a program that gives a computer the capability to communicate with internet and display the information stored there. e.g operamini,google chrome,internet explorer. The first page of a website is what we call the homepage.

To create a basic web page we need the following:

  1. Html
  2. Css

STRUCTURE OF A WEB PAGE

  • html- This tag is used to start the description of the web page.it should be closed at the end of the web page.
  • head- It is used to enclose header information that usually does not appear in the browser window.
  • title- This is used to specify what is displayed on the title bar of the browser.
  • body- This contain the main content of the website
