HTML LINKS: One defining feature of HTML document is the ability to move from webpage to another using links. HTML links are hyperlinks. Hyperlinks are specially formatted webpage objects (text or image) that a user can click to jump to (open) another document either on the same webpage or other websites. The world wide web got it spidery name from the plentiful connections (links) that link websites together with the click of a button. The use of hyperlinks make creation of extensive documents easier where section or topics can be quickly accessed through links. Anchor Element The HTML anchor element is used to create a link or hyperlink reference (href) to a resource such as another web page, a file, a multimedia element, and so on or to a specific place within a web page. Each hyperlink begins with an tag and ends with an tag. The opening and closing tags surround the text to click to perform hyperlink. The anchor tag requires some attribute and value to work.

Hypertext Reference(href): The beginning anchor tag takes a very important attribute called the HREF (Hypertext Reference). The href attribute defines reference that the link refers to. It specifies the destination address. This is where the user will be taken if they wish to click this link. Hypertext references can be Internal, Local, or Global.
